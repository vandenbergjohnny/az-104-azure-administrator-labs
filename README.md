# AZ-104 Azure Administrator Labs

This repository documents my hands-on preparation for the Microsoft AZ-104 Azure Administrator certification.

The labs focus on practical Azure administration skills including identity, governance, networking, compute, storage, monitoring, backup, and migration readiness.

## Focus Areas

- Microsoft Entra ID users and groups
- Role-Based Access Control
- Azure Policy and governance
- Virtual networks, subnets, NSGs, and routing
- Azure Virtual Machines
- Azure Storage Accounts
- Azure Backup and Recovery Services Vault
- Azure Monitor and Log Analytics
- Azure CLI and PowerShell
- Azure migration readiness for server workloads

## Why I Created This Repository

I am building practical Azure administration and cloud migration skills to support enterprise Azure environments and large-scale server migration projects.

My background includes enterprise Control-M administration, workload automation, production support, Managed File Transfer, Linux, Windows Server, and hybrid infrastructure operations.
